# XF 项目经验

## 🏛️ 智慧政务与企业数字化

### 1. 某市网上政务大厅与智慧政务系统

**技术栈**：Vue.js, React, Native H5/CSS3, RemarkUI
**项目描述**：响应政府“互联网+政务服务”号召构建的一站式在线审批与监管平台，旨在推动行政效能提升与数字化转型。
**核心职责**：
*   **多技术栈融合**：在项目中灵活运用 **Vue** 与 **React** 开发不同业务模块，实现了在线咨询、数字化审批流程、数据共享交换等核心功能。
*   **公众服务门户**：负责构建高兼容性的政务大厅前端，优化了多渠道接入体验（PC/Mobile），让企业与公民能便捷地办理行政业务。
*   **数据可视化**：实现了政务数据的可视化展示，辅助政府部门进行实时监管与决策。

### 2. 企业级云办公与智能门禁系统

**技术栈**：React, Vue, Node.js, NW.js, WebSocket
**项目描述**：一套集成了云办公（OA）、考勤管理与硬件互联的企业应用生态系统，覆盖桌面端、移动端与 Web 端。
**核心职责**：
*   **跨平台桌面应用**：利用 **NW.js** 技术构建了跨平台的企业桌面客户端，实现了与 Web 端功能的无缝同步。
*   **软硬互联**：基于 **WebSocket** 实现了实时通信服务，支撑**微信扫码开门**等 IoT 场景，打通了小程序、手机客户端与门禁硬件的数据链路。
*   **全栈开发**：参与了部分 **Node.js** 后端服务开发，配合 React 构建了功能完备的后台管理系统。

## 🛍️ 电商零售与垂直行业系统

### 3. 红酒商城 ERP 与微信分销系统

**技术栈**：React, RemarkUI, 微信小程序
**项目描述**：集成了产品管理、销售 CRM、ERP 库存管理与微信裂变分销的综合零售系统。
**核心职责**：
*   **复杂业务逻辑实现**：负责后台管理系统中产品、订单、会员等核心模块的开发，实现了前端与后端 ERP 系统的深度数据对接。
*   **移动销售终端**：开发了面向门店销售人员的**微信小程序**，支持移动端开单、库存查询与业绩统计，提升了线下销售效率。
*   **分销体系构建**：实现了基于微信生态的分销裂变功能，通过多级分销机制助力产品推广。

### 4. "超级领队" 旅游管理系统

**技术栈**：React, jQuery, JSP, HTML/CSS
**项目描述**：面向旅游行业的垂直管理平台，包含面向游客的 Web 端与面向领队的后台管理系统。
**核心职责**：
*   **遗留系统维护与新功能开发**：在维护基于 **JSP + jQuery** 的传统 Web 端页面的同时，引入 **React** 技术栈重构后台管理系统。
*   **前后端协同**：配合后端工程师优化了页面交互行为，确保了从前台展示到后台配置的数据一致性。
