# What is an AI Agent?

> "An intelligent agent is a system that perceives its environment and takes actions that maximize its chances of success."

- **Perception**: Inputs (Text, Files, API responses)
- **Brain**: LLM (Large Language Model)
- **Action**: Tools (Search, Database, Code Execution)

---

# Why Monorepo?

- **Unified Codebase**: Frontend and Backend in one place.
- **Type Safety**: Share TypeScript interfaces between Client and Server.
- **Efficiency**: Turborepo caches build artifacts.
