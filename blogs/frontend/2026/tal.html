# TAL 项目经验

# 🚀 核心业务项目

# 1. 云平台日志中心与监控系统

技术栈：Vue 3, TypeScript, Vite 4, Ant Design 4, UnoCSS, qiankun (微前端) 项目描述：全公司统一的日志查询、链路追踪与监控告警平台，支撑每日 TB 级日志量的检索与分析。 核心职责：

  • 架构重构与性能优化：主导项目从 Vue 2 + Webpack 向 Vue 3 + Vite + UnoCSS 的架构迁移，构建速度提升 300%+，首屏加载时间减少 40%。
  • 微前端体系建设：基于 qiankun 搭建微前端架构，将日志中心、站内信、埋点管理等多个子应用整合成统一的云平台门户，实现了基座与子应用的解耦开发与独立部署。
  • 复杂可视化交互：针对海量日志场景，实现了基于虚拟滚动 (Virtual Scrolling) 的高性能列表渲染，支持 10万+ 行数据的流畅浏览；封装了通用的图表组件库，支持多维度的日志数据可视化分析。
  • 工程化规范落地：制定并推广前端工程化规范（ESLint, Prettier, Husky），输出前端代码工程规范open in new window等技术文档，提升团队协作效率。

# 2. 低代码问卷搭建平台

技术栈：Vue 3, TypeScript, NestJS, Ant Design, JSON Schema 项目描述：一款企业级在线问卷调查与数据收集平台，支持通过拖拽方式快速生成复杂表单。 核心职责：

  • 低代码引擎设计：设计了一套基于 JSON Schema 的表单描述协议，实现了可视化拖拽编辑器，支持组件的动态配置、联动逻辑与校验规则，大幅降低了问卷创建门槛。
  • 全栈开发：使用 NestJS 搭建后端服务，实现了问卷数据的增删改查（CRUD）、用户权限管理及数据统计接口，打通了前后端全链路开发。
  • 提效成果：将问卷页面的开发周期从天级缩短至分钟级，有效支撑了公司内部各类信息收集需求。

# 3. API 开发管理与运维平台

技术栈：Vue 2, Ant Design, ECharts, JSON Schema 项目描述：包含 API 管理、数据质量监测、运维监控在内的多个中后台管理系统。 核心职责：

  • 标准化脚手架建设：从零搭建了组内首个基于 Vue + Ant Design 的标准化项目模板，统一了目录结构与基础库版本，使新项目启动时间缩短 50%。
  • 配置化组件封装：设计并封装了 FormList（配置化表单） 和 SearchTable（配置化搜索表格） 高阶组件，通过 JSON 配置即可生成 80% 的中后台页面，极大地提高了开发效率。
  • 数据服务化：实现了 API 服务的快速搭建与编排，优化了数据流动流程，提升了数据质量监测的实时性与准确性。

# 4. 数据查询分析系统 (Data Query Analysis)

技术栈：Vue 2, Element UI, ECharts, SQL Compiler 项目描述：面向内部数据分析师与运营人员的自助式数据查询与报表平台，服务于公司内部 600+ 活跃用户。 核心职责：

  • SQL 编译器维护：维护前端 SQL 编译器模块，支持复杂的 SQL 语法解析、智能补全与语法高亮，同时支持无 SQL 的可视化查询模式，降低了用户使用门槛。
  • 高性能图表渲染：实现了通用数据匹配与图表展示功能，支持折线图、柱状图、散点图等多种图表的动态切换；利用虚拟列表技术解决了大量查询结果（万级数据）的前端渲染卡顿问题。
  • 稳定性保障：优化了查询请求的调度与缓存策略，显著提升了系统的查询响应速度与稳定性，保障了高并发场景下的用户体验。
